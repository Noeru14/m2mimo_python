	* DESCRIPTION DES DOSSIERS & FICHIERS

- GO : contient le code source en python ;

- CDC : cahier des charges ;

- templates : contient le fichier index.html ;

- static : contient les fichiers js (appelé depuis index.html), xml, img et css ;

- __init__.py : permet de lancer le projet python sous console ;

- serveurgo.py : permet de lancer le projet en chargeant le fichier index.html.
